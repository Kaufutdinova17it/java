# Отчет по лабораторной работе

**Вариант:** Нечётный

Лабораторная работа разделена на 4 блока по 5 задач. Каждый блок и задачи в нем реализованы с помощью метода `switch` и кейсов.

## Блок 1

### Задача 1
**Сигнатура метода:** `public double fraction(double x)`

**Задача:** Вернуть только дробную часть числа x

**Решение:** 
- На вход подается вещественное число
- С помощью `int` записываем целую часть числа
- Из вещественного вычитаем целую часть
- Получаем дробную часть

### Задача 3
**Сигнатура метода:** `public int charToNum(char x)`

**Задача:** Преобразовать символ в соответствующее число

**Решение:**
- На входе получаем один из символов '0'-'9'
- Вычитаем из него символ '0'
- Получаем нужное число

### Задача 5
**Сигнатура метода:** `public bool is2Digits(int x)`

**Задача:** Проверить, является ли число двузначным

**Решение:**
- Проверяем находится ли число в диапазоне от 10 до 99
- Возвращаем `true` если да, `false` если нет

### Задача 7
**Сигнатура метода:** `public bool isInRange(int a, int b, int num)`

**Задача:** Проверить вхождение числа в диапазон (включая границы)

**Решение:**
- Сравниваем какая граница больше/меньше
- Проверяем, что число больше меньшей границы и меньше большей границы

### Задача 9
**Сигнатура метода:** `public bool isEqual(int a, int b, int c)`

**Задача:** Проверить равенство трех чисел

**Решение:**
- Проверяем что первое число равно второму, а второе третьему
- Возвращаем `true` если все равны, `false` если нет

## Блок 2

### Задача 1
**Сигнатура метода:** `public int abs(int x)`

**Задача:** Вернуть модуль числа

**Решение:**
- Если число меньше 0 - возвращаем противоположное число
- Если число больше 0 - возвращаем само число

### Задача 3
**Сигнатура метода:** `public bool is35(int x)`

**Задача:** Проверить делимость на 3 или 5 (но не одновременно)

**Решение:**
- Проверяем делимость на 3 ИЛИ на 5
- Исключаем случай одновременной делимости
- Возвращаем `true` если условия выполняются

### Задача 5
**Сигнатура метода:** `public int max3(int x, int y, int z)`

**Задача:** Найти максимальное из трех чисел

**Решение:**
- Присваиваем x значение max
- Последовательно сравниваем с y и z
- Обновляем max при необходимости

### Задача 7
**Сигнатура метода:** `public int sum2(int x, int y)`

**Задача:** Вернуть сумму чисел (с особым условием)

**Решение:**
- Складываем числа x и y
- Если сумма в диапазоне 10-19 - возвращаем 20
- Иначе возвращаем сумму

### Задача 9
**Сигнатура метода:** `public String day(int x)`

**Задача:** Вернуть день недели по номеру

**Решение:**
- Используем `switch` для каждого дня недели
- 1 - понедельник, 7 - воскресенье
- При неверном вводе возвращаем "это не день недели"

## Блок 3

### Задача 1
**Сигнатура метода:** `public String listNums(int x)`

**Задача:** Вернуть строку с числами от 0 до x

**Решение:**
- Создаем пустую строку для результата
- Цикл от 0 до x
- Добавляем числа к строке результата

### Задача 3
**Сигнатура метода:** `public String chet(int x)`

**Задача:** Вернуть строку с четными числами от 0 до x

**Решение:**
- Создаем пустую строку для результата
- Цикл от 0 до x с шагом 2
- Добавляем четные числа к строке

### Задача 5
**Сигнатура метода:** `public int numLen(long x)`

**Задача:** Вернуть количество цифр в числе

**Решение:**
- Если x = 0 - сразу возвращаем 1
- Создаем счетчик и копию числа
- Цикл while: делим на 10 и увеличиваем счетчик
- Возвращаем количество цифр

### Задача 7
**Сигнатура метода:** `public void square(int x)`

**Задача:** Нарисовать квадрат из символов '*'

**Решение:**
- Внешний цикл для количества строк
- Внутренний цикл для символов в строке
- Выводим '*' x раз в каждой строке

### Задача 9
**Сигнатура метода:** `public void rightTriangle(int x)`

**Задача:** Нарисовать треугольник, выровненный по правому краю

**Решение:**
- Внешний цикл для количества строк
- Внутренний цикл для вывода пробелов и звездочек
- Количество звездочек равно номеру строки

## Блок 4

### Задача 1
**Сигнатура метода:** `public int findFirst(int[] arr, int x)`

**Задача:** Найти индекс первого вхождения числа

**Решение:**
- Проходим по массиву циклом
- Сравниваем каждый элемент с x
- Возвращаем индекс при совпадении
- Если не найдено - возвращаем -1

### Задача 3
**Сигнатура метода:** `public int maxAbs(int[] arr)`

**Задача:** Найти число с наибольшим модулем

**Решение:**
- Берем первый элемент как начальный максимум
- Проходим по массиву циклом
- Сравниваем модули чисел
- Запоминаем число с наибольшим модулем

### Задача 5
**Сигнатура метода:** `public int[] add(int[] arr, int[] ins, int pos)`

**Задача:** Вставить массив в указанную позицию

**Решение:**
- Проверяем корректность позиции
- Создаем новый массив нужного размера
- Копируем элементы до позиции вставки
- Вставляем элементы второго массива
- Копируем оставшиеся элементы

### Задача 7
**Сигнатура метода:** `public int[] reverseBack(int[] arr)`

**Задача:** Вернуть перевернутый массив

**Решение:**
- Проверяем массив на null
- Создаем новый массив той же длины
- Заполняем в обратном порядке
- Возвращаем результат

### Задача 9
**Сигнатура метода:** `public int[] findAll(int[] arr, int x)`

**Задача:** Найти все индексы вхождения числа

**Решение:**
- Первый проход: считаем количество совпадений
- Создаем массив нужного размера
- Второй проход: записываем индексы совпадений
- Возвращаем массив индексов
